This is a public, sanitized export generated from a local results workspace.

Sanitization policy:

- Private IP addresses, MAC addresses, link-local IPv6 addresses, local paths, usernames, hostnames, MACSec SCI values, and credential-like assignments were redacted.
- CSV payload columns such as `value_hex` were replaced with `REDACTED_PAYLOAD` while keeping performance metrics and counts.
- PNG figures were kept with their top title/header band covered because original plot titles exposed network identifiers.
- PCAP/PCAPNG files, raw payload dumps, binary captures, archives, compiled artifacts, certificates/keys, and local virtual environments were not copied raw. They are represented by placeholders or manifest entries.
- The original workspace was not modified; this directory is the generated public export.

# Thesis Context Needed

La información del repositorio permite documentar herramientas, resultados y metodología experimental. Para dejar el repositorio totalmente alineado con la tesis, faltan estos datos de contexto académico:

- Título exacto de la tesis.
- Objetivo general y objetivos específicos.
- Preguntas de investigación o hipótesis.
- Descripción formal de la topología experimental.
- Hardware usado en SCADA, PLC/API, puente y host de medición.
- Sistema operativo y versiones relevantes.
- Parámetros MACsec que puedan publicarse sin revelar claves.
- Criterios de aceptación: latencia máxima tolerable, pérdida permitida, consumo esperado.
- Número de repeticiones por escenario y justificación estadística.
- Convención final de nombres para escenarios y corridas.
- Formato de citación deseado para el repositorio.

Con esos datos, el README puede convertirse en una introducción académica completa y los documentos pueden mapearse directamente a capítulos de metodología, resultados y reproducibilidad.
